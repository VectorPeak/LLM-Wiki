一、深度学习早期的注意力机制（2014–2017）

1、Bahdanau Attention（Additive Attention）  
（1）首次将注意力引入神经机器翻译（NMT），发表于 ICLR 2015；  
（2）使用前馈网络计算 query 与 key 的相似度：  
  score(h\_i, s\_j) = v^T tanh(W\_1 h\_i + W\_2 s\_j)；  
（3）解决 RNN 编码器信息瓶颈问题，实现软对齐。

2、Luong Attention（Multiplicative / Dot-Product Attention）  
（1）提出三种变体：dot、general、concat；  
（2）dot-product 形式为 QK^T，计算更高效；  
（3）区分 global（全序列）与 local（局部窗口）注意力。

3、Self-Attention（自注意力雏形）  
（1）在《A Structured Self-Attentive Sentence Embedding》（ICLR 2017）中用于句子表示；  
（2）通过 attention 权重加权词向量，生成上下文感知的句嵌入；  
（3）尚未形成多头、位置编码等完整架构。

二、Transformer 时代（2017–2022）

1、Scaled Dot-Product Attention  
（1）出自《Attention Is All You Need》（NeurIPS 2017）；  
（2）公式：Attention(Q,K,V) = softmax(QK^T / √d\_k) V；  
（3）引入缩放因子 √d\_k 防止点积过大导致 softmax 梯度消失。

2、Multi-Head Attention（MHA）  
（1）并行多个注意力头，捕捉不同子空间的交互模式；  
（2）输出拼接后经线性变换：Concat(head\_1,...,head\_h)W^O；  
（3）成为 Transformer Encoder/Decoder 的核心模块。

3、Positional Encoding  
（1）使用正弦/余弦函数注入绝对位置信息；  
（2）支持任意长度外推（理论上）；  
（3）后续被可学习位置嵌入、相对位置编码等替代。

4、Relative Position Representations  
（1）Shaw et al.（NAACL 2018）：在 attention logits 中加入可学习的相对位置偏置；  
（2）Transformer-XL（2019）：引入 segment-level recurrence + relative positional encoding，支持长文本建模。

三、大模型时代（2022 至今）

1、Rotary Position Embedding（RoPE）  
（1）Su et al.（2021 提出，2022 后广泛采用）；  
（2）将位置信息编码为旋转矩阵，作用于 Q/K 向量；  
（3）天然支持相对位置，具备良好长度外推能力；  
（4）被 Llama、Qwen、ChatGLM 等主流开源模型采用。

2、FlashAttention 系列  
（1）FlashAttention（2022）：通过分块计算+IO 感知算法，减少 HBM 访问，加速 attention；  
（2）FlashAttention-2（2023）：优化 warp 调度，进一步提升吞吐；  
（3）FlashAttention-3（2024）：支持 FP8 与结构化稀疏，适配 H100。

3、稀疏注意力（Sparse Attention）  
（1）Longformer（2020）：滑动窗口 + 全局 attention；  
（2）BigBird（2020）：随机 + 滑动 + 全局连接，理论可证明表达能力；  
（3）Mistral（2023）：Sliding Window Attention（SWA），仅关注局部上下文，高效支持 32K。

4、GQA（Grouped-Query Attention）  
（1）Google（2023）提出，介于 MHA 与 MQA 之间；  
（2）将多个 query 共享同一组 key/value，减少 KV Cache 显存；  
（3）Llama-2-70B、Mixtral、Qwen 等采用，平衡速度与质量。

5、MQA（Multi-Query Attention）  
（1）所有 head 共享同一组 K/V，大幅压缩缓存；  
（2）推理速度提升显著，但可能损失部分表达能力；  
（3）用于 StarCoder、Falcon 等模型。

6、ALiBi（Attention with Linear Biases）  
（1）Press et al.（2021）；  
（2）不在输入中加位置编码，而是在 attention logits 上减去与距离成正比的偏置；  
（3）天然支持任意长度外推，被 Bloom、MPT 采用。

7、MLA（Multi-head Latent Attention）  
（1）DeepSeek-V2（2024）提出；  
（2）将 K/V 压缩为低维潜在表示，减少显存带宽压力；  
（3）适用于 MoE 架构下的高吞吐推理。

8、门控注意力（Gated Attention）  
（1）NeurIPS 2025 最佳论文提出；  
（2）在 attention 输出上增加 sigmoid 门控，引入非线性；  
（3）缓解“首 token 注意力陷阱”，提升训练稳定性。

四、其他重要变体与思想

1、Cross-Attention  
（1）用于 Encoder-Decoder 架构，query 来自 decoder，key/value 来自 encoder；  
（2）是 seq2seq、RAG、扩散模型中的关键组件。

2、Causal / Masked Attention  
（1）通过下三角 mask 实现自回归生成；  
（2）标准 Decoder-only 模型的基础。

3、Memory-Efficient Attention  
（1）如 xFormers、PagedAttention（vLLM）；  
（2）优化显存布局，支持高并发长上下文推理。

4、Ring Attention / Sequence Parallelism  
（1）将长序列切分到多个 GPU，突破单卡显存限制；  
（2）支持百万级 token 训练（如 LongNet、Yarn）。
