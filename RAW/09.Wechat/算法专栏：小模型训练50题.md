12. 12\. 使用 vLLM 部署小模型时，PagedAttention 的 block 管理开销可能超过收益。分析当模型层数 且上下文长度 时，传统 contiguous KV Cache 与 PagedAttention 的内存访问效率差异。

15. 15\. 在多任务小模型训练中，不同任务的梯度可能冲突。推导 PCGrad (Projected Gradient Descent) 方法中梯度投影操作 对优化轨迹的影响。
16. 16\. 当小模型用于边缘设备时，需进行算子融合以减少 kernel launch 开销。描述如何将 LayerNorm、Linear 和 GELU 融合成单个 CUDA kernel，并分析其对端到端延迟的提升。
17. 17\. 在使用 DeepSpeed 的 activation checkpointing 时，对于层数较少的小模型（如 6 层），重新计算所有激活的 FLOPs 开销可能超过显存收益。推导最优 checkpointing 策略的 break-even point。
18. 18\. 分析小模型在 FP16 训练中更容易出现梯度下溢（underflow）的现象。从梯度幅值的统计分布出发，推导其与模型深度和初始化尺度的关系，并提出一种动态 loss scaling 策略。
19. 19\. 在指令微调小模型时，若指令模板过于复杂，会挤占有效上下文。设计一种基于 instruction compression 的数据预处理方法，在保持语义不变的前提下最小化 token 消耗。
20. 20\. 使用 Ring Attention 扩展小模型上下文窗口是否合理？从通信开销 与计算开销 的比值出发，分析其在小模型（ 小）长上下文（ 大）场景下的适用性。
21. 21\. 在小模型中集成稀疏激活（如 Top-2 MoE）时，若专家数量过多，会导致专家利用率低下。推导专家激活概率的熵 与路由负载均衡的关系，并设定一个最小利用率阈值。
22. 22\. 对比直接量化（PTQ）与量化感知训练（QAT）在小模型上的精度损失。从 Hessian-aware 量化的角度，解释为何小模型对权重扰动更敏感。
23. 23\. 在小模型的 RLHF 训练中，Reward Hacking 现象更严重。分析其成因：即小模型无法充分建模复杂奖励函数，从而找到高奖励但语义错误的捷径，并提出一种基于 reward uncertainty 的正则化项。
24. 24\. 使用 FSDP 训练小模型时，若模型总参数量小于单卡显存，分片反而引入通信开销。推导 FSDP 的 all-gather 通信量与模型大小、GPU 数量的关系，并给出关闭分片的阈值条件。
25. 25\. 在小模型中使用 ALiBi 位置编码而非 RoPE，分析其对训练稳定性和外推能力的影响。特别地，推导 ALiBi 偏置项对梯度范数的抑制效应。
26. 26\. 当小模型作为 Agent 的策略网络时，其探索能力有限。设计一种基于参数噪声（parameter noise）的探索机制，并分析其与 action noise 在 sample efficiency 上的差异。
27. 27\. 在多语言小模型训练中，低资源语言易被高资源语言淹没。推导一种基于 language identity embedding 的平衡采样策略，使其能根据语言难度动态调整采样概率。
28. 28\. 分析小模型在使用 RMSNorm 而非 LayerNorm 时的训练动态差异。从梯度流的角度，解释为何 RMSNorm 能减少小模型训练中的不稳定现象。
29. 29\. 在小模型的 DPO 训练中，若偏好数据质量不高（如含噪声配对），分析其对决策边界的影响。推导一个基于 pair-wise 一致性的鲁棒 DPO 变体。
30. 30\. 使用混合精度训练小模型时，若 master weights 以 FP32 存储，optimizer states 占用仍较大。设计一种针对 Adam 的 8-bit optimizer states 压缩方案，并分析其对收敛性的影响。
31. 31\. 在小模型的蒸馏过程中，若教师模型输出存在系统性偏差（如过度自信），分析其对学生模型 calibration 的负面影响，并提出一种 temperature annealing 策略。
32. 32\. 对比小模型在 CPU 与 NPU 上的推理性能。分析算子支持度、内存带宽和量化格式（如 INT8 vs. INT4）对端到端延迟的贡献权重。
33. 33\. 在小模型中使用 SwiGLU 激活函数而非 ReLU，推导其对梯度传播和表示能力的影响。特别地，分析其在低维空间中的表达效率。
34. 34\. 当小模型用于在线学习时，频繁更新会导致参数震荡。设计一种基于弹性权重巩固（EWC）的在线微调算法，保护已学知识。
35. 35\. 在小模型的多任务训练中，若任务相关性低，硬参数共享会导致负迁移。推导一种基于 task-specific LoRA 的软共享架构，并分析其参数效率。
36. 36\. 分析小模型在长尾分类任务中的偏差放大现象。从 logits 分布的协方差结构出发，推导一种解耦表征学习（decoupled representation learning）策略。
37. 37\. 在小模型的 RL 训练中，价值函数估计方差大。推导一种基于 n-step return 和 GAE 的混合优势估计方法，并分析其在小模型上的 bias-variance trade-off。
38. 38\. 使用模型合并（Model Merging）技术集成多个小模型时，若各模型在不同子任务上表现优异，设计一种基于任务感知的权重插值策略。
39. 39\. 在小模型的联邦学习中，客户端异构性更显著。推导一种个性化 LoRA 适配器的聚合规则，使其能在全局共享与本地定制间取得平衡。
40. 40\. 分析小模型在使用 dynamic batching 时的吞吐瓶颈。推导最优 batch size 与序列长度方差、GPU 利用率的关系。
41. 41\. 在小模型的对比学习预训练中，负样本数量受限。推导 memory bank 机制对梯度估计方差的降低效果，并分析其与 MoCo 的等价性。
42. 42\. 当小模型用于生成任务时，beam search 易产生重复。分析其与模型 capacity 的关系，并提出一种基于 coverage penalty 的改进解码策略。
43. 43\. 在小模型的训练中，若使用 cosine learning rate decay，推导其在有限步数下的收敛界，并与 linear decay 进行比较。
44. 44\. 分析小模型在使用 grouped convolution 替代 dense linear layer 时的表达能力损失。从 Johnson-Lindenstrauss 引理出发，推导 group 数与信息保留率的关系。
45. 45\. 在小模型的多模态训练中，模态对齐更困难。设计一种基于 contrastive learning 的跨模态对齐损失，并分析其在小 batch size 下的稳定性。
46. 46\. 使用 speculative decoding 加速小模型推理时，draft model 与 target model 的容量差距会影响接受率。推导最优 draft model size 与 target model 的关系。
47. 47\. 在小模型的 continual pretraining 中，新旧领域分布偏移大。推导一种基于 domain-adversarial training 的领域自适应策略。
48. 48\. 分析小模型在使用 weight tying（输入输出嵌入共享）时的性能增益。从参数效率和梯度一致性角度解释其在小模型上的必要性。
49. 49\. 在小模型的 RLHF 训练中，若 reward model 也是小模型，其评分噪声大。设计一种基于 ensemble 的 reward smoothing 机制。
50. 50\. 使用 tensor parallelism 训练小模型是否合理？从通信开销 与计算开销 的比值出发，推导其适用的模型规模下限。
