```python
def findAnagrams(self, s: str, p: str) -> List[int]:
    ans = []
    cnt = Counter(p)
    left = 0
    for right, c in enumerate(s):
        cnt[c] -= 1  # 右端字母进入窗口
        while cnt[c] < 0:  # 字母c太多了
            cnt[s[left]] += 1  # 左端字母离开窗口
            left += 1
        if right - left + 1 == len(p):  # s和p的每种字母的出现次数相同
            ans.append(left)
    return ans
```

#### 1.5 前后缀最大值模板

```python
def productExceptSelf(self, nums: List[int]) -> List[int]:
    n = len(nums)
    pre = [1] * n
    for i in range(1, n):
        pre[i] = pre[i-1] * nums[i-1]

    suf = [1] * n
    for i in range(n-2, -1, -1):
        suf[i] = suf[i+1] * nums[i+1]

    return [p * s for p, s in zip(pre, suf)]
```

#### 1.6 单调队列模板
**降本增效**：维护单调递减队列
```python
def maxSlidingWindow(self, nums: List[int], k: int) -> List[int]:
    ans = [0] * (len(nums) - k + 1)
    q = deque()
    for right, x in enumerate(nums):
        # 1. 右边入
        while q and nums[q[-1]] <= x:  # 新进入的数大于最后一个数，维护单调递减
            q.pop()
        q.append(right)  # 注意是保存下标，这样可以判断是否离开窗口

        # 2. 左边出
        left = right - k + 1
        if q[0] < left:
            q.popleft()

        # 3. 在窗口左端点记录答案
        if left >= 0:
            ans[left] = nums[q[0]]
```

#### 1.7 动态规划模板
**核心思想**：避免重复计算和最优化子结构

**状态定义** → **状态转移方程** → **边界条件**

**启发思路**：选或不选 / 选哪个

##### 方法一：记忆化搜索（自顶向下）
```python
def rob(self, nums: List[int]) -> int:
    n = len(nums)
    cache = [-1] * n  # 用 cache 来做记忆化

    def dfs(i):
        if i < 0:
            return 0
        if cache[i] != -1:
            return cache[i]
        res = max(dfs(i-1), dfs(i-2) + nums[i])
        cache[i] = res
        return res

    return dfs(n-1)
```
##### 方法二：递推（自底向上）
```python
def rob(self, nums: List[int]) -> int:
    n = len(nums)
    f = [0] * (n + 2)
    for i, x in enumerate(nums):
        f[i+2] = max(f[i+1], f[i] + x)  # 这里因为避免出现负数所以 i+2
    return f[n+1]
```

#### 1.8 链表模板
**什么时候需要用到哨兵节点？**
如果需要删除头节点的时候是比较合适的，注意一定是删除的时候才需要。

##### 相交链表
```python
def getIntersectionNode(self, headA: ListNode, headB: ListNode) -> Optional[ListNode]:
    p, q = headA, headB
    while p is not q:
        p = p.next if p else headB
        q = q.next if q else headA
    return p
```
##### 反转链表
```python
def reverseList(self, head: Optional[ListNode]) -> Optional[ListNode]:
    pre = None
    cur = head
    while cur:  # cur指向None
        nxt = cur.next
        cur.next = pre
        pre = cur
        cur = nxt
    return pre  # pre是新的头节点
```
##### 链表中点
```python
def middleNode(self, head: Optional[ListNode]) -> Optional[ListNode]:
    slow = fast = head
    while fast and fast.next:
        slow = slow.next
        fast = fast.next.next
    return slow
```
##### 回文链表
```python
def isPalindrome(self, head: Optional[ListNode]) -> bool:
    mid = self.middleNode(head)
    head2 = self.reverseNode(mid)
    while head2:
        if head.val != head2.val:
            return False
        head = head.next
        head2 = head2.next
    return True
```
##### 删除倒数第N个节点
```python
def removeNthFromEnd(self, head: Optional[ListNode], n: int) -> Optional[ListNode]:
    dummy = left = right = head = ListNode(next=head)
    for _ in range(n):
        right = right.next
    while right.next:
        left = left.next
        right = right.next
    left.next = left.next.next  # 删除链表节点的操作
    return dummy.next
```
##### 两两交换节点
```python
def swapPairs(self, head: Optional[ListNode]) -> Optional[ListNode]:
    if not head or not head.next:
        return head
    newhead = head.next
    head.next = self.swapPairs(newhead.next)
    newhead.next = head
    return newhead
```
##### 获取链表长度

```python
def getListLength(self, head: Optional[ListNode]) -> int:
    length = 0

        length += 1
        head = head.next
```

### 二、哈希表专题
#### 1. 两数之和
**题目**：给定一个整数数组 `nums` 和一个整数目标值 `target`，请你在该数组中找出和为目标值的那两个整数，并返回它们的数组下标。
```python
class Solution:
    def twoSum(self, nums: List[int], target: int) -> List[int]:
        record = {}
        for index, num in enumerate(nums):
            complement = target - num
            if complement in record:
                return [record[target-num], index]
            record[num] = index
        return []
```
**核心要点**：

- 使用字典（`{}`）而不是列表（`[]`），因为字典查找时间复杂度为 O(1)
- 字典存储的是键值对，可以通过「键」快速查找「值」

#### 49. 字母异位词分组
**题目**：给你一个字符串数组，请你将字母异位词组合在一起。
```python
class Solution:
    def groupAnagrams(self, strs: List[str]) -> List[List[str]]:
        d = defaultdict(list)
        for s in strs:
            sorted_s = ''.join(sorted(s))  # 把s排序，作为hashmap的key
            d[sorted_s].append(s)  # 排序后相同的字符串分到同一组
        return list(d.values())
```
**核心要点**：

- `defaultdict(list)`：访问不存在的键时，会自动把该键存入字典，值设为空列表
- `Counter`：访问不存在的键时，只返回 0，但不会把该键存入字典
- 如果后续需要遍历字典的所有键，`defaultdict(list)` 会包含那些只是看了一眼但没修改的键

#### 128. 最长连续序列
**题目**：给定一个未排序的整数数组 `nums`，找出数字连续的最长序列的长度。
```python
class Solution:
    def longestConsecutive(self, nums: List[int]) -> int:
        st = set(nums)
        ans = 0
        for num in st:
            if num - 1 in st:
                continue
            y = num + 1
            while y in st:
                y += 1
            ans = max(ans, y - num)  # 一定是y-num，这样才是长度
        return ans
```
**核心要点**：

- 只从序列的起点开始计数（`num-1` 不在集合中）
- `ans` 不用 `+=1`，直接计算长度

### 三、双指针专题
#### 167. 两数之和 II - 输入有序数组
**题目**：给你一个下标从 1 开始的整数数组 `numbers`，该数组已按非递减顺序排列，请你从数组中找出满足条件的那两个数。
```python
class Solution:
    def twoSum(self, numbers: List[int], target: int) -> List[int]:
        left = 0
        right = len(numbers) - 1
        while left < right:
            s = numbers[left] + numbers[right]
            if s == target:
                break
            if s > target:
                right -= 1
            else:
                left += 1
        return [left + 1, right + 1]
```

#### 15. 三数之和
**题目**：给你一个整数数组 `nums`，判断是否存在三元组满足 `nums[i] + nums[j] + nums[k] == 0`。
```python
class Solution:
    def threeSum(self, nums: List[int]) -> List[List[int]]:
        ans = []
        n = len(nums)
        nums.sort()

        for i in range(n - 2):
            if i > 0 and nums[i] == nums[i-1]:  # i>0保证第一个数能取到
                continue

            x = nums[i]
            if x + nums[i + 1] + nums[i + 2] > 0:  # 优化一
                break
            if x + nums[-2] + nums[-1] < 0:  # 优化二
                continue

            j = i + 1
            k = n - 1
            while j < k:
                s = nums[i] + nums[j] + nums[k]
                if s > 0:
                    k -= 1
                elif s < 0:
                    j += 1
                else:
                    ans.append([nums[i], nums[j], nums[k]])
                    j += 1
                    while j < k and nums[j] == nums[j-1]:
                        j += 1
                    k -= 1
                    while j < k and nums[k] == nums[k+1]:
                        k -= 1
        return ans
```
**核心要点**：

- `i > 0` 保证第一个数能取到，避免忽略全零数列
- 三元组的顺序不重要，答案中不可包含重复的三元组

#### 16. 最接近的三数之和
```python
class Solution:
    def threeSumClosest(self, nums: List[int], target: int) -> int:
        nums.sort()
        ans = -inf
        n = len(nums)
        for i in range(n - 2):
            x = nums[i]
            left = i + 1
            right = n - 1
            while left < right:
                s = nums[left] + nums[right]
                if s + x < target:
                    left += 1

                    right -= 1
                else:

                if abs(ans - target) > abs(s + x - target):
                    ans = s + x
```

```python
class Solution:

        n = len(nums)
```

#### 13. 盛水最多的容器
**核心思路**：指针每一次移动，都意味着排除掉了一个柱子。小的移动。
```python
class Solution:
    def maxArea(self, height: List[int]) -> int:
        ans = 0
        left = 0
        right = len(height) - 1
        while left < right:
            area = (right - left) * min(height[left], height[right])
            ans = max(area, ans)
            if height[left] < height[right]:
                left += 1
            else:
                right -= 1
        return ans
```

#### 42. 接雨水
##### 方法一：前后缀最大值
```python
class Solution:
    def trap(self, height: List[int]) -> int:
        n = len(height)
        pre_max = [0] * n
        pre_max[0] = height[0]
        for i in range(1, n):
            pre_max[i] = max(pre_max[i-1], height[i])

        suf_max = [0] * n
        suf_max[-1] = height[-1]
        for i in range(n-2, -1, -1):
            suf_max[i] = max(suf_max[i+1], height[i])

        ans = 0
        for h, pre, suf in zip(height, pre_max, suf_max):
            ans += min(pre, suf) - h
        return ans
```
##### 方法二：双指针优化

如果前缀最大值比后缀最大值小，那么左边木桶的容量就是前缀最大值，算完之后左指针右移一位。
```python
class Solution:
    def trap(self, height: List[int]) -> int:
        n = len(height)
        left = 0
        right = n - 1
        ans = 0
        pre_max = 0
        suf_max = 0
        while left <= right:
            pre_max = max(pre_max, height[left])
            suf_max = max(suf_max, height[right])
            if pre_max < suf_max:
                ans += pre_max - height[left]
                left += 1
            else:
                ans += suf_max - height[right]
                right -= 1
        return ans
```

#### 125. 验证回文串
```python
class Solution:
    def isPalindrome(self, s: str) -> bool:
        i, j = 0, len(s) - 1
        while i < j:
            if not s[i].isalnum():  # 若左指针字符无效（非字母/数字），左指针右移
                i += 1
            elif not s[j].isalnum():
                j -= 1
            elif s[i].lower() == s[j].lower():  # 若两者都是有效字符，转为小写后比较
                i += 1
                j -= 1
            else:  # 不相等：直接返回 False
                return False
        return True
```

```python
class Solution:
```

```python
        ans = 0
        cnt = Counter()  # hashmap key=char value=int
        left = 0
        for right, x in enumerate(s):
```

#### 438. 找到字符串中所有的字母异位词
##### 方法一：定长滑动窗口
```python
class Solution:
    def findAnagrams(self, s: str, p: str) -> List[int]:
        left = 0
        ans = []
        cnt_s = Counter()
        cnt_p = Counter(p)
        for right, c in enumerate(s):
            cnt_s[c] += 1
            left = right - len(p) + 1
            if left < 0:
                continue
            if cnt_p == cnt_s:  # s和p的每种字母的出现次数都相同
                ans.append(left)
            cnt_s[s[left]] -= 1
        return ans
```
##### 方法二：不定长滑动窗口
```python
class Solution:
    def findAnagrams(self, s: str, p: str) -> List[int]:
        ans = []
        cnt = Counter(p)
        left = 0
        for right, c in enumerate(s):
            cnt[c] -= 1  # 右端字母进入窗口
            while cnt[c] < 0:  # 字母c太多了
                cnt[s[left]] += 1  # 左端字母离开窗口
                left += 1
            if right - left + 1 == len(p):  # s和p的每种字母的出现次数相同
                ans.append(left)
        return ans
```

#### 76. 最小覆盖子串
```python
class Solution:
    def minWindow(self, s: str, t: str) -> str:
        cnt_s = Counter()
        cnt_t = Counter(t)

        ans_left = -1
        ans_right = len(s)
        left = 0
        for right, c in enumerate(s):
            cnt_s[c] += 1
            while cnt_s >= cnt_t:  # 涵盖
                if right - left < ans_right - ans_left:
                    ans_left, ans_right = left, right
                cnt_s[s[left]] -= 1
                left += 1
        return "" if ans_left < 0 else s[ans_left:ans_right+1]
```

#### 209. 长度最小的子数组
**滑动窗口指针模板**：

- **最大值**：判定条件（不满足），在外面更新 ans
- **最小值**：判定条件（满足），在里面更新 ans
```python
class Solution:
    def minSubArrayLen(self, target: int, nums: List[int]) -> int:
        ans = inf
        left = 0
        s = 0
        n = len(nums)
        for right, x in enumerate(nums):
            s += x
            while s >= target:  # 满足要求
                ans = min(ans, right - left + 1)  # 满足要求在里面更新最小值
                s -= nums[left]
                left += 1
        return ans if ans <= n else 0
```
**问**：为什么不能用「选或不选 nums[i]」的思路做？
**答**：选或不选无法保证子数组是连续的。选或不选适用于子序列问题（例如 0-1 背包问题），对于子数组问题，更适合用「拼接」的思路，即如果 nums[i] 左边的子数组元素和是负的，就不用和左边的子数组拼在一起了。

#### 713. 乘积小于 K 的子数组
**如何计算右端点固定下的子数组个数？**
```python
left ... right
[l,r] [l+1,r] ... [r,r]
最终的子数组个数就是 r-l+1
class Solution:
    def numSubarrayProductLessThanK(self, nums: List[int], k: int) -> int:
        if k <= 1:
            return 0
        left = 0
        ans = 0
        s = 1
        for right, x in enumerate(nums):
            s *= x
            while s >= k:  # 最大值判定条件不满足要求，在外面更新ans
                s = s / nums[left]
                left += 1
            ans += (right - left + 1)  # 对于固定的right，有right-left+1个合法的左端点
        return ans
```
