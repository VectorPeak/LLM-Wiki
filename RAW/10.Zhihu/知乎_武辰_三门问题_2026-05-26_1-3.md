## 一、有趣的数学——违反直觉的囚犯问题和三门问题

监狱里关了三个囚犯A、B、C。监狱长决定赦免其中一个人，并且完全随机地选择了一人作为赦免人（另外两个人将被处死）。他将自己的选择告诉了看守，但是要求看守暂时不能泄露被赦免人的姓名。

囚犯A问看守“谁被赦免了”，看守拒绝回答。A接着问道：“B和C之中谁一定会被处死？”

看守心想：“不论谁被赦免，B和C之中一定有一个人会被处死。因此我告诉他这个问题的答案也无妨。”于是看守说B会被处死。当然，看守不会说谎。

囚犯A心想：“A一开始被赦免的概率是三分之一，现在我知道B不是被赦免的那个人，于是只有A和C是有可能被赦免的人，因此A被赦免的概率提升到了二分之一。”

A的分析对吗？

你可以设身处地地假想自己是A并在脑海里走一遍这个过程。大多数人都会觉得A被赦免的概率变大了。然而事实是：A被赦免的概率没有发生变化，相反，囚犯C被赦免的概率增加到了三分之二。

这个问题和广为人知的“三门问题”的本质是一致的。关于三门问题，我有文章进行详细的讨论：

[武辰：[终结数学月经题系列2]三门问题](https://zhuanlan.zhihu.com/p/36094978)

不过在这篇文章里，我打算用数学语言来分析这个问题。尽管用到数学公式，我将尽力用通俗易懂的方式解释清楚每一步。这个问题在著名的统计学教材《Statistical Inference》里有提及。我保证以下问题分析的答案是正确的，请读者放心。

---
以下解答过程涉及到概率论中最基本的条件概率公式与全概率公式，没有用到更深的数学知识。只假定读者具有最基础的概率论知识储备。

我们记：

事件 $A$ =“A被赦免”，

事件 $B$ =“B被赦免”，

事件 $C$ =“C被赦免”，

事件 $W$ =“看守说B将被处死”。

由条件概率公式：

$P(A|W)=\frac{P(A \cap W)}{P(W)}$ （1）

我们要算的是 $P(A|W)$ ，即：“看守说B将被处死”的条件下，A是被赦免人的概率。要算这个概率，我们只需算等式右边的概率。等式右边看起来更复杂，但是它是可以被算出来的。

我们首先算分母。由全概率公式：

$P(W)=P(A)P(W|A)+P(B)P(W|B)+P(C)P(W|C)$ （2）

（事实上如果把（2）式代入（1）式，式子就变成了大名鼎鼎的贝叶斯公式）

首先显然 $P(A)=P(B)=P(C)=\frac{1}{3}$ 。

$P(W|A)$ 表示A被赦免的条件下，看守说B被处死的概率。由于如果A是被赦免人，看守既可以说B被处死，也可以说C被处死，而B和C的地位是相等的，因此这个概率是 $\frac{1}{2}$ 。

$P(W|B)$ 表示B被赦免的条件下，看守说B被处死的概率。这是矛盾的，所以这个概率为0。

$P(W|C)$ 表示C被赦免的条件下，看守说B被处死的概率。由于C被赦免，那么B一定会被处死，所以这个概率为1。

把上面的六个概率代入（2）式，得 $P(W)=\frac{1}{2}$ 。

同时我们已经求出 $P(A \cap W)=P(A)P(W|A)=\frac{1}{6}$ ，所以再代入（1）式，得

$P(A|W)=\frac{P(A \cap W)}{P(W)}=\frac{1}{3}$ 。

即看守回答B将被处死后，囚犯A被赦免的概率仍然是 $\frac{1}{3}$ ，而非 $\frac{1}{2}$ 。同时更神奇的是，囚犯C被赦免的概率增加到了 $\frac{2}{3}$ ，因为 $P(C|W)=1-P(A|W)-P(B|W)=1-\frac{1}{3}-0=\frac{2}{3}$

我们回头分析为什么囚犯A（和我们大多数人）的推理是错的，以及这个问题反直觉的地方在哪里。根本原因在于大多数人把事件 $W$ 等同于事件 $\bar{B}$ （ $\bar{B}$ 指的是事件$B$的对立事件）。

我们首先设定了事件 $W$ =“看守说B将被处死”，事件 $B$ =“B将被赦免”，那么事件 $\bar{B}$ =“B将被处死”。

问题的核心在于：**“看守说B将被处死”与“B将被处死”两句话看似一样，其实包含了不同的信息**。我们从对立事件的角度就能分析出背后的奥妙。

事件$W$ （即：看守说B将被处死）的对立事件是“看守说C将被处死”，而事件$\bar{B}$ （即：B将被处死）的对立事件是“B将被赦免”。两个事件的对立事件是不同的。**第一个事件着重回答“谁将被处死”，第二个事件的重点在于“B是否会被处死”**。而囚犯A错误地将事件 $W$ 等同于事件 $\bar{B}$ ，由此他产生了如下推理：

$P(A|\bar{B})=\frac{P(A \cap \bar{B})}{P(\bar{B})}$ （正如之前的分析，这个式子与（1）式的区别在于把 $W$ 都替换成了 $\bar{B}$ ）。

由于 $P(\bar{B})=\frac{2}{3}$ ， $P(A \cap \bar{B})=P(A)P(\bar{B}|A)=\frac{1}{3}\cdot1=\frac{1}{3}$

所以 $P(A|\bar{B})=\frac{P(A \cap \bar{B})}{P(\bar{B})}=\frac{1/3}{2/3}=\frac{1}{2}$

错误答案$\frac{1}{2}$ 就是这样产生的。

---
事实上，“三门问题”与这道题是等价的。

“三个囚犯”对应于“三扇门”，

“囚犯被赦免”对应于“门后面有车”，

“你是囚犯A”对应于“你一开始选择了第一扇门”，

“看守说B会被处死”对应于“第二扇门后面没有车”，

“看守说B会被处死后，囚犯A被赦免的概率不会变，囚犯C被赦免的概率增加到了 $\frac{2}{3}$”对应于“在主持人展示出第二扇门后面没有车后，你获奖的概率不会变，但是第三扇门获奖的概率增加到了 $\frac{2}{3}$ ，所以你应该选择换门”。

因此“三门问题”也可以通过类似的严格的数学语言解释。

“囚犯问题”和“三门问题”本质上是一个问题，也是困扰很多人的问题。网上已经有很多对这个问题的讨论，但尽管如此，仍有不少人对这个问题模棱两可、不知其所以然，处于云里雾里的状态。究其原因在于网上对三门问题的讨论多是出于“概率思维”的直观解释。所以我在此用严格的数学语言完整地证明了结论，希望这篇文章能够让读者对这两个问题有清晰的理解。
## 二、[终结数学月经题系列2]三门问题

所谓月经话题，是指论坛里几乎每个月都会出现的话题性与争议性很强的问题。比如“星期二男孩问题”：

[武辰：[终结数学月经题系列1]星期二男孩问题](https://zhuanlan.zhihu.com/p/36071292)

本文讲“三门问题”：

有一个电视游戏竞猜节目，主持向竞猜者展示了三扇门。有一扇门之后是一辆小轿车，另两扇门之后是空房间。主持人事先知道门后是什么。

游戏分为三步：首先，你选择一扇门；然后，主持人将会打开剩余的两扇门中的一扇，展示一个空的房间（当然他从未打开那扇后面藏有轿车的门）；最后你可以选择是仍然选择你开始选择的那扇门，还是选择去打开另一扇仍然关闭的门。 如果你选择换门，则你获得小轿车的概率是多少？

咋一看，貌似换不换门都无所谓。因为不管你选择哪扇门，主持人都会打开一扇空门，主持人打开空门这个动作看起来是多余的。

事实上，这个主持人是**上帝视角**。假设你一开始想打开A门，然后主持人打开了C门，展示C门是空的。你有一次机会，要不要改成B门？

这个时候你需要想想，为什么主持人**偏偏不打开B这扇门呢**？ （主持人并不是随机打开门的，而是只打开后面没有车的门）

我举一个极端的例子。假设有100个门，只有一个有汽车，其他都是空的。不妨设你一开始想开第一扇门（或者如果你想打开第77扇门，那么我们将门重新编号，把原来的第77扇门编号成第1扇门，以此类推……这就是“不妨”的意思），那么你获得小汽车的概率只有百分之一。

这个时候，假设主持人打开了第2-36扇门和第38-100扇门，他一共打开了98扇门，且门后都是空的。这个时候问你，要不要换成打开第37扇门？

当然要换！第一扇门背后有汽车的概率只有百分之一，而第37扇门背后有车的概率是百分之99！

我们回过头想想，为什么主持人偏偏要绕过第37扇门？这背后大概率有猫腻。也就是说，37号门背后大概率有车。

我们再回到三扇门的情况。一开始ABC背后有车的概率都是三分之一，你想打开A门。在主持人打开C门之后，样本空间发生了改变，B背后有车的概率从三分之一变到三分之二。

所以问题的关键是：**观察者透露出多少信息**。在星期二男孩问题中，他是只知道第一个孩子的信息呢，还是已经观察了所有孩子的信息，这将决定问题的样本空间。

这个问题和“囚犯问题”是等价的。数学证明在这篇文章中：

## 三、还在纠结三门问题？不如用代码暴力验证（附Python代码）

三门问题是一个经常在各大论坛里出现的问题。对于大众来说，三门问题可以说是最具有富有争议性的数学问题之一。三门问题具体表述如下：

有一个电视游戏竞猜节目，主持向挑战者（也就是你）展示了三扇门。有一扇门之后是一辆小轿车，另两扇门之后是空房间。主持人事先知道门后是什么。

游戏分为三步：首先，你选择一扇门；然后，主持人将会打开剩余的两扇门中的一扇，展示一个空的房间（当然他从未打开那扇后面藏有轿车的门）；最后，你可以选择是仍然选择你开始选择的那扇门，还是选择去打开另一扇仍然关闭的门。 如果你选择换门，则你获得小轿车的概率是多少？

咋一看，貌似这道题的答案是二分之一。因为不管你选择哪扇门，主持人都会一边向你坏笑一边打开一扇空门，主持人打开空门这个动作看起来是多余的。

但是答案是：选择换门是更好的策略。理论证明可以见我的这篇文章：

[武辰：[终结数学月经题系列2]三门问题](https://zhuanlan.zhihu.com/p/36094978)

如果你不想看理论证明，或者不认同我的答案，或者看完理论解释后仍然云里雾里，怎么办？亲自用程序模拟一遍或许对你有帮助。

在下文，我给出了我的编程模拟的过程。我的代码是用Python写的，关键的地方都加了注释。

![image](https://pic2.zhimg.com/v2-ab33ca669d7cf82704867b6d76849b6d_1440w.jpg)
```python
import random
buhuan=0 #不换
huan=0 #换

for i in range(10000): #这意味着我们要做1000次实验
    car = random.choice([1,2,3]) #汽车随机出现在三扇门之中
    challenger = random.choice([1,2,3]) #挑战者随机选择了一扇门
    #以下五行代码：主持人随机选择一扇门，这扇门不是挑战者选择的门，也不是汽车所在的门
    host_list = [1,2,3]
    host_list.remove(challenger)
    if car in host_list:
        host_list.remove(car)
    host = random.choice(host_list)

    if challenger == car:
        buhuan = buhuan+1 #如果挑战者一开始选的门背后有汽车，那么不换是对的
    else:
        huan = huan+1 #如果挑战者一开始选的门背后没有汽车，那么换是对的

print(buhuan)
print(huan)
```
代码运行的结果是：换门相比于不换门，获奖的次数多了一倍！

---
开个脑洞，把题目稍微做变动：主持人事先不知道车在哪个门后面。如果主持人在第二步打开的门刚好是有汽车的门，那么游戏重来。在这种情况下，换门与不换门孰优孰劣？
```python
import random
buhuan=0
huan=0

for i in range(1000):#这意味着我们要做1000次实验
    car = random.choice([1,2,3])#汽车随机出现在三扇门之中
    challenger = random.choice([1,2,3])#挑战者随机选择了一扇门
    #以下三行代码：主持人随机选择一扇门，这扇门不是挑战者选择的门
    host_list = [1,2,3]
    host_list.remove(challenger)
    host = random.choice(host_list)

    if host == car:
        continue #如果主持人在第二步选的门是汽车，则重来

    if challenger == car:
        buhuan = buhuan+1
    else:
        huan = huan+1

print(buhuan)
print(huan)
```
代码运行的结果是：换门与不换门的效果是一样的。

---
那么问题又来了，假如我不知道主持人事先是否知道车在哪扇门后面，那么挑战者应该换门吗？

这个问题只能定性分析，而无法定量分析。答案当然是应该换门。因为不论主持人事先是否知道车在哪扇门后面，换门都至少不比不换门差。但是我不知道换门后获得汽车的具体概率，而且我认为没有人能够求出这个概率，如果大家都是忠实的频率学派的拥护者的话。事实上并不是所有的事情都是有概率可言的。所以如果要精确求概率的话，这个问题就变得无意义了。
