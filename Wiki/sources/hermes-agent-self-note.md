# 八股生存指北：Hermes

## 摘要

这篇自笔记从面试角度整理 Hermes Agent，重点关注它与 OpenClaw 的相似性和差异，尤其是记忆系统、自我进化循环、自动 skill 沉淀和 Agent 架构。

## 关键断言

- Hermes 的核心面试点集中在记忆模块、自我进化和与 OpenClaw 的区别。
- 记忆系统可从静态结构、动态更新、压缩、反思和 skill 沉淀几个角度解释。
- Hermes 被笔记视为热门 Agent 产品案例，适合用于展示对行业趋势的理解。
