# 深度学习入门：基于 Python 的理论与实现

## 摘要

这本书是深度学习入门教材，强调用 Python 从零实现经典神经网络。它适合作为理解神经网络、反向传播、CNN 和训练技巧的基础来源。

## 关键断言

- 从零实现可以降低对框架黑箱的依赖。
- 神经网络基础、反向传播和卷积网络是后续学习 NLP 与 LLM 的地基。
