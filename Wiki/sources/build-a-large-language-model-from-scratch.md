# 从零构建大模型

## 摘要

这本书以从零实现大模型为主线，覆盖模型组件、数据准备、预训练、微调、指令跟随和加载预训练权重等内容。它适合连接深度学习基础和现代 LLM 工程。

## 关键断言

- 理解 LLM 不能只停留在 API 层，需要掌握 tokenizer、Transformer、训练目标和微调流程。
- 从零实现能帮助读者理解大模型各组件如何协同。
