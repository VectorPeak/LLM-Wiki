# Agent 面试准备框架

## 方法定位

Agent 面试准备框架用于把散乱题目组织成可回答、可追问、可举例的知识结构。它不只是背定义，而是训练候选人说明系统边界和工程取舍。

## 七类问题

1. Agent 架构与核心组件。
2. 推理与规划范式。
3. Memory 系统设计。
4. Tool Calling 与函数调用。
5. 多 Agent 协作。
6. RAG 系统核心问题。
7. 生产可靠性与容错恢复。

## 回答模板

每个问题优先按四步回答：定义、机制、工程坑、项目例子。面试官继续追问时，再补充指标、替代方案和边界条件。
