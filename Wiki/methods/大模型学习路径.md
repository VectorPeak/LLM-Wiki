# 大模型学习路径

## 顶层路线

1. 数学、Python 和算法基础。
2. [[神经网络基础]] 与 [[反向传播]]。
3. NLP 前置知识：[[Word2Vec]]、[[RNN]]、语言模型。
4. [[Transformer]]。
5. [[大模型预训练与微调]]。
6. RAG、Agent、工具调用和生产系统。

## 学习原则

大模型学习应同时走两条线：底层原理线和工程应用线。只学原理容易脱离生产，只做应用容易停留在 API 调用。
