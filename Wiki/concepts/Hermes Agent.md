# Hermes Agent

## 定义

Hermes Agent 是使用者自笔记中整理的 Agent 产品案例，重点被放在记忆系统、自我进化循环和自动 skill 沉淀上。

## 笔记中的核心结构

- 触发层、网关层、Agent 层。
- Context Window、工作空间文件、session transcript、retrieval index、user model。
- 写入、读取、压缩和反思策略。
- 自进化循环与 skill 自我改进。

## 与 OpenClaw 的对比线索

笔记把 Hermes 与 OpenClaw 视为同一类产品，差异点集中在记忆系统和自我进化能力。这个判断需要后续联网核对。
